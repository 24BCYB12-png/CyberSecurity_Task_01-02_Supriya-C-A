# Cyber Security Task 01: Personal Security Audit & Cyber Awareness Assessment

## Overview

This repository folder contains my submission for **Cyber Security Task 01**, a personal security audit and cyber awareness assessment completed as part of the cybersecurity internship. The purpose of this task was to assess my own digital security practices, identify potential risks, and build foundational cyber awareness — since cybersecurity starts with securing yourself before securing systems and networks.

This is a learning and self-assessment task and is not marks-based.

## Key Findings from Part A (Device Security Assessment)

- **Operating System:** Windows 11 Pro, Version 25H2 (OS Build 26200.8655)
- **Update Status:** System is up to date (last checked same day)
- **Antivirus:** Enabled — no current threats found, security intelligence up to date

## Key Takeaway

This task reinforced that cybersecurity begins with personal habits — not just tools. Reviewing my own passwords, accounts, and system settings helped me identify real gaps (such as missing MFA on one account) and apply practical fixes immediately, rather than treating cybersecurity as a purely theoretical topic.
